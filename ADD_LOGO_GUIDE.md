# Adding the Squaawke Logo

## 📸 Steps to Add Your Logo

### 1. Create Assets Folder

```bash
cd "/Volumes/WD Elements -1/ace/acestep-local"
mkdir -p assets
```

### 2. Save the Logo Image

Save your logo image as: `assets/squaawke-logo.png`

**Option A: Manual**
- Save the image file to your desktop
- Move it to the `assets/` folder
- Rename it to `squaawke-logo.png`

**Option B: Command Line**
```bash
# If you have the image file already
cp /path/to/your/logo.png assets/squaawke-logo.png
```
